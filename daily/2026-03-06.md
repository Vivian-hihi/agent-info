## 行业动态

### Google A2A 协议
Google 于 2025 年 4 月发布 Agent2Agent（A2A）开放协议，目标是让不同厂商的 Agent 互通。核心亮点：
- 与 Anthropic MCP 互补（MCP 管 Agent ↔ 工具，A2A 管 Agent ↔ Agent）
- 基于 HTTP/SSE/JSON-RPC，易于集成
- 支持长时任务、多模态
- 50+ 合作伙伴（Salesforce、SAP、LangChain 等）

详见 → [multi-agent/a2a-protocol.md](../multi-agent/a2a-protocol.md)
